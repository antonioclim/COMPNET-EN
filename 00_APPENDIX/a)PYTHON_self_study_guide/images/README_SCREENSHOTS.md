# 📸 Screenshots Folder

This folder is reserved for screenshots that accompany the guide.

## Naming Convention

```
[step]_[topic]_[description].png
```

Examples:
- `01_socket_server_terminal.png`
- `02_wireshark_tcp_capture.png`
- `03_portainer_dashboard.png`

## Adding Screenshots

1. Capture with `Win + Shift + S` (Windows) or `gnome-screenshot` (Linux)
2. Save with descriptive name following convention
3. Reference in markdown:

*(Screenshots to be added. Place `.png` files in this directory and reference them here.)*
