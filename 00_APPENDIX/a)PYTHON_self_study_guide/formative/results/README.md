# Quiz Results Directory

Output directory for quiz session results generated by `run_quiz.py`. The `.gitkeep` file ensures Git tracks this otherwise-empty folder; actual result files are git-ignored.

| File | Description |
|---|---|
| [`.gitkeep`](.gitkeep) | Placeholder to preserve the directory in version control |

No quiz result files are committed to the repository. Each student's local runs populate this directory with session-specific JSON or plain-text summaries.

## Cross-References

The quiz runner that writes to this directory lives at [`../run_quiz.py`](../run_quiz.py). Question definitions are in [`../quiz.yaml`](../quiz.yaml).

**Suggested sequence:** complete the quiz via `make quiz` in [`../../`](../../) → results appear here.
