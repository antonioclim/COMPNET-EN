# Seminar 02 — Socket programming: TCP/UDP server and clients

This seminar covers the implementation of concurrent TCP and UDP servers together with their corresponding clients. Traffic is analysed with Wireshark to observe the differences between connection-oriented and connectionless communication.

## Contents

Scenario files:

- `S02_Part03_Scenario_TCP_Server_Netcat_Wireshark.md` — TCP server tested with netcat and Wireshark
- `S02_Part11_Scenario_UDP_Server_Client_Wireshark.md` — UDP server and client with traffic capture

The seminar includes 9 Python source files, 11 HTML support pages in `../_HTMLsupport/S02/` and 4 PlantUML diagrams in `assets/puml/`.
