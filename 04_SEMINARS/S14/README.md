# Seminar 14

Seminar 14 is a consolidation session for revision and the team project assessment.

It provides three components:

* an explanation of the assessment model, the evidence expectations and the viva format
* a student preparation checklist and tasks
* a rubric template that can be used by assessors and by teams for self-audit

This seminar assumes you are working on one of the projects in `02_PROJECTS/`.
Common deliverable standards and deterministic verification are documented in:

* `02_PROJECTS/00_common/README_STANDARD_RC2026.md`

## Contents

| File | Purpose |
|---|---|
| `S14_Part01_Explanation_Revision_and_Assessment.md` | Assessment model and revision guidance |
| `S14_Part02_Tasks_Preparation.md` | Preparation tasks and self-checks |
| `S14_Part03_Template_Assessment_Rubric.md` | Rubric, evidence checklist and viva prompts |
