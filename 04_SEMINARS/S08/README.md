# Seminar 08 — HTTP server and Nginx reverse proxy

Students implement a minimal HTTP server from raw sockets, compare it with Python's built-in HTTP server module and then configure Nginx as a reverse proxy using Docker Compose.

- `S08_Part02A_Explanation_Http_server_Builtin.md` — Python built-in HTTP server
- `S08_Part03A_Explanation_HTTP_Socket.md` — HTTP server from sockets
- `4_nginx/S08_Part04A_Explanation_Nginx_Reverse_Proxy.md` — Nginx reverse proxy setup
