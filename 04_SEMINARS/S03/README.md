# Seminar 03 — UDP broadcast/multicast and TCP tunnel

This seminar extends socket programming to group communication patterns (broadcast, multicast and anycast over UDP) and introduces a simple TCP tunnel for forwarding traffic between endpoints.

## Contents

Scenario files:

- `S03_Part03_Scenario_TCP_Multi_client_Server.md` — multi-client TCP server
- `5_udp-multicast/S03_Part05D_Scenario_UDP_Multicast.md` — UDP multicast communication
- `6_udp-anycast/S03_Part06D_Scenario_UDP_Anycast.md` — UDP anycast behaviour

The seminar includes 12 Python source files, 6 HTML support pages in `../_HTMLsupport/S03/` and 5 PlantUML diagrams in `assets/puml/`.
