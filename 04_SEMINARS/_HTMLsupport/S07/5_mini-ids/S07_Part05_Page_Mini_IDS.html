How this Mini IDS Works

This simulation visualises the Python logic from mini_ids.py.

1. TCP Port Scan Detection

Logic: The IDS tracks SYN packets (connection requests) that do not have the ACK flag set.

  • Key: Source IP.
  • Value: Set of distinct Destination Ports accessed within WINDOW_SECONDS.
  • Alert: Triggered if the count of distinct ports ≥ TCP_SYN_THRESHOLD (10).

2. UDP Spray Detection

Logic: Since UDP is connectionless, we simply count unique destination ports contacted by a source.

  • Key: Source IP.
  • Value: Set of distinct Destination Ports accessed within WINDOW_SECONDS.
  • Alert: Triggered if the count of distinct ports ≥ UDP_PORT_THRESHOLD (10).

3. Flood Detection (DoS)

Logic: Detects high traffic volume to a single target service.

  • Key: Tuple (Destination IP, Destination Port).
  • Value: List of packet timestamps.
  • Alert: Triggered if packet count > FLOOD_THRESHOLD within the time window.
