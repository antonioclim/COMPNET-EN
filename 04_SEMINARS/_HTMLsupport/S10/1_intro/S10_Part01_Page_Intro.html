Lab Guide

1. DNS Investigation

Pending

Computers communicate via IP addresses. DNS translates human names (like example.com) into IPs.

$ dig example.com $ dig A example.com
Goal: Observe the ANSWER SECTION and TTL.

2. Secure Connection

Pending

Establish a secure encrypted tunnel to manage a remote server.

$ ssh student@192.168.1.10
Goal: Complete the cryptographic handshake.

3. Remote Operations

Locked

Execute commands on the remote machine.

$ uname -a $ ls -la
