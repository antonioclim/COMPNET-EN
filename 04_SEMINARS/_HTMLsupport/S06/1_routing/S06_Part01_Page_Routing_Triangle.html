Lab Tasks

1. Verify IP Configuration

Check interfaces on all nodes.

r1 ip a r2 ip a r3 ip a

2. Configure Indirect Path

Set routes via r2 (r1->r2->r3).

r1 ip route add ... via r2 r2 ip route add ... via r3 r2 return route to h1 r3 return route via r2

3. Test Connectivity

Ping from h1 to h3.

h1 ping -c 3 10.0.3.2

4. Trace the Path

Verify which path is taken.

h1 traceroute 10.0.3.2

5. Switch Route (Direct)

Change r1 to go directly to r3.

Delete current route Add route via 10.0.13.2 (r3) Return path via r1 (Direct)
