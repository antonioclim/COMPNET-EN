What is Os-Ken?
Os-Ken (formerly Ryu) is the "brain" of the network (Control Plane).
  • Decision: Runs the Python script that decides which packets pass.
  • Protocol: Communicates with the switch via OpenFlow.
  • Action: When s1 receives unknown traffic, it sends it here. Os-Ken analyses it and sends back rules (Flow-Mod).
