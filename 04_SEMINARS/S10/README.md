# Seminar 10 — DNS, SSH and FTP in Docker

Network services are deployed and orchestrated inside Docker containers. Students configure a DNS resolver, set up SSH client-server communication with Paramiko, implement SSH port forwarding and interact with an FTP service — all within containerised environments.

- `S10_Part02A_Explanation_DNS_Containers.md` — DNS service in Docker
- `S10_Part03A_Explanation_SSH_Paramiko.md` — SSH with Paramiko
- `S10_Part04A_Explanation_SSH_Portforward.md` — SSH port forwarding

Task files:
