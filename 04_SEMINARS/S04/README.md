# Seminar 04 — Custom text and binary protocols

Students design and implement application-level protocols: a text-based protocol over TCP, a binary protocol over TCP using struct packing and a UDP-based protocol. Each variant demonstrates different trade-offs in serialisation, framing and error handling.

## Contents

Scenario files:

- `1_text-proto_tcp/S04_Part01D_Scenario_Text_Proto_TCP.md` — text protocol over TCP
- `2_binary-proto_tcp/S04_Part02D_Scenario_Binary_Proto_TCP.md` — binary protocol over TCP
- `3_proto_udp/S04_Part03D_Scenario_UDP_Proto.md` — protocol over UDP

The seminar includes 13 Python source files, 3 HTML support pages in `../_HTMLsupport/S04/` and 5 PlantUML diagrams in `assets/puml/`.
