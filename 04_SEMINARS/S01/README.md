# Seminar 01 — Network analysis: Wireshark, netcat and traffic debugging

This seminar introduces fundamental network diagnostic tools. Students capture and inspect traffic with Wireshark and tshark, use netcat for raw TCP and UDP communication and learn systematic approaches to debugging network behaviour.

## Contents

Scenario files:

- `S01_Part01_Scenario_Basic_Tools.md` — basic command-line network tools
- `S01_Part03_Scenario_Netcat_Basics.md` — netcat TCP and UDP sessions
- `S01_Part05_Scenario_Wireshark_Netcat.md` — packet capture while using netcat

Task files:

- `S01_Part02_Tasks_Basic_Tools.md`
- `S01_Part04_Tasks_Netcat_Basics.md`
- `S01_Part06_Tasks_Wireshark_Netcat.md`
