# 00_common/assets/images — Rendered Diagram Output

Placeholder directory for PNG files produced by [`../render.sh`](../render.sh) from the PlantUML sources in [`../puml/`](../puml/). Currently contains only `.gitkeep` to preserve the directory in version control.

Run `bash ../render.sh` to populate this folder.

## Selective Clone

```bash
git sparse-checkout set 02_PROJECTS/00_common/assets/images
```
